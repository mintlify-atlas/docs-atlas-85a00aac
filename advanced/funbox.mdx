Funbox modes are special gameplay modifiers that add unique challenges and variations to your typing tests. They range from visual effects to completely different typing mechanics.

## What is Funbox?

Funbox modes transform your standard typing test into creative and challenging variations. Some funboxes are purely cosmetic, while others fundamentally change how you type. Each funbox has a difficulty rating and some allow you to save personal bests while others don't.

<Note>
Funbox modes can be combined with different test configurations, but some combinations may be restricted based on the funbox properties.
</Note>

## Visibility Funboxes

These modes challenge your memory and reading speed by hiding or limiting visible words.

<Accordion title="Read Ahead Modes">
  - **Read Ahead Easy**: Only the current word is invisible (Difficulty: 1/3)
  - **Read Ahead**: Current and the next word are invisible (Difficulty: 2/3)
  - **Read Ahead Hard**: Current and the next two words are invisible (Difficulty: 3/3)
  
  All read ahead modes save personal bests and force highlight mode to "off" or "letter".
</Accordion>

<Accordion title="Plus Modes">
  - **Plus Zero**: React quickly! Only the current word is visible (Difficulty: 1/3)
  - **Plus One**: Only one future word is visible (Difficulty: 0/3)
  - **Plus Two**: Only two future words are visible (Difficulty: 0/3)
  - **Plus Three**: Only three future words are visible (Difficulty: 0/3)
  
  These modes cannot be used with infinite duration tests.
</Accordion>

<Accordion title="Memory">
  Test your memory. Remember the words and type them blind.
  
  **Difficulty:** 3/3 | **Saves PB:** Yes
  
  Memory mode shows you all the words briefly, then hides them. You must type from memory. Only works with words, quote, or custom modes.
</Accordion>

## How to Use Funbox

1. Navigate to the test configuration area
2. Look for the funbox icon or menu
3. Select one or more compatible funboxes
4. Start your test with the modified rules

<Note>
Some funboxes cannot be combined with certain test modes or configurations. The interface will prevent incompatible combinations.
</Note>

## Personal Bests

Funboxes marked with "Saves PB: Yes" allow you to save personal best scores. These modes are considered legitimate challenges. Funboxes that don't save PBs are typically practice modes or significantly alter the mechanics in ways that make scores incomparable to standard tests.
